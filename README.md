# Fast multipole methods
## Draft of master's thesis in fluid mechanics

Credits: 60 ECTS

Author: Simon Lederhilger

Institution: Department of Mathematics, Faculty of Mathematics and Natural Sciences,  University of Oslo

## Compilation

Compiling the thesis from draft/ using TeX Live:
```sh
latexmk main.tex
```

Compiling figures from the figures/ directory:
```sh
latexmk -cd this_figure/this_figure.tex
```
Figures are precompiled to save time compiling main.
