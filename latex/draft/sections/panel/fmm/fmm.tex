The fast multipole method calculates the velocity at some point $\bzhe$ using a sum of values using the methods of equation (?) for near field $\xvec$ and values using the multipole method.
A local coordinate system is adopted, requiring translation and rotation of the origin, at which point a multipole expansion can be made.
We suppose a dipole panel
\[
  \Phi(\bzhe) = -\frac{1}{4\pi} \int_S \frac{\mathbf{\mu} \cdot \mathbf{r}}{r^3} \,\dee S,
\]
where $\mathbf{\mu}$ is the dipole strength vector.
